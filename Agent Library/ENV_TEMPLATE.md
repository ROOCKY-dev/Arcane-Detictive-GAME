# Environment Variables

Create a `.env.local` file in the project root with these values:

```env
# Supabase Configuration
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here

# Optional: Ad network
NEXT_PUBLIC_AD_CLIENT_ID=your-ad-client-id

# Optional: Analytics
NEXT_PUBLIC_ANALYTICS_ID=your-analytics-id

# App Configuration
NEXT_PUBLIC_APP_NAME=SQL Quest: The Realm of Syntaxia
NEXT_PUBLIC_APP_URL=https://sqlquest.vercel.app
```
