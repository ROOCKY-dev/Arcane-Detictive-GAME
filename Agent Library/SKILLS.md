# Required Skills & Technologies

## Core Technologies
- **TypeScript** — Primary language for all code
- **React.js 18+** — UI framework
- **Next.js 14+** — React meta-framework (App Router)
- **Tailwind CSS** — Utility-first CSS framework
- **sql.js** — SQLite compiled to WebAssembly for in-browser SQL execution
- **CodeMirror 6** — Extensible code editor for the SQL terminal
- **Framer Motion** — Animation library

## Backend & Infrastructure
- **Supabase** — Auth, PostgreSQL database, real-time subscriptions
- **Vercel** — Hosting and deployment
- **GitHub** — Version control and CI/CD

## SQL Knowledge (for content creation)
- SELECT, FROM, WHERE, AND/OR
- ORDER BY, LIMIT
- JOIN (INNER, LEFT)
- GROUP BY, HAVING
- COUNT(), SUM(), AVG(), MIN(), MAX()
- Subqueries
- LIKE, IN, BETWEEN
- DISTINCT
- INSERT, UPDATE, DELETE (for advanced cases)

## Build & Development Tools
- **Node.js 18+** — Runtime
- **pnpm** or **npm** — Package manager
- **ESLint** — Linting
- **Prettier** — Code formatting

## Database Tooling
- **SQLite3** (CLI or DB Browser for SQLite) — for creating/inspecting .sqlite seed files
- **Custom build script** (`scripts/build-databases.ts`) — generates .sqlite files from .sql seeds

## Optional / Nice-to-Have
- **Howler.js** — Sound effects library
- **Canvas Confetti** — Celebration effects on case solved
- **Recharts** — Charts for teacher analytics dashboard
- **React Hot Toast** — Toast notifications
