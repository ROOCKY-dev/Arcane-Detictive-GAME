## Important Notes
- Generate all backgrounds at **1920x1080** (16:9 landscape)
- Generate NPC portraits at **512x768** or **3:4 ratio** (portrait orientation)
- Generate icons at **512x512** then downscale to 128x128 in the app
- All icons should have **black/transparent backgrounds** for easy compositing
- Use the **same AI model** for all assets to maintain consistency
- If using Midjourney, add `--ar 16:9` for landscapes, `--ar 3:4` for portraits
- After generation, use a consistent post-processing pass (same brightness/contrast/saturation)
